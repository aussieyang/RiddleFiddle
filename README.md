# Riddle Fiddle

Welcome to Riddle Fiddle!

Access the game at:
http://riddlefiddle.herokuapp.com/

The game is designed for 1 host on a large screen and 4 players on mobile devices. The game commences as soon as 4 players join a room.

### Tech Stack:

- NodeJS
- ExpressJS
- Socket.IO
